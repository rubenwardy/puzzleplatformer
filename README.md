Puzzle Platformer
=================

A platforming game and engine written using Javascript, HTML5 and CSS.


Included Libraries
------------------

* [Crafty.js](http://craftyjs.com/)
* [Box2D.js](http://box2d.org/)
* Modified version of [CraftyBox2D](https://github.com/shogoki-vnz/CraftyBox2D)
